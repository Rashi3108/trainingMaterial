What is Java?

‚òï

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle) in 1995. It's designed to be platform-independent, meaning "write once, run anywhere."

Key Features of Java

  • Platform Independent: Java code runs on any system with JVM
  • Object-Oriented: Everything in Java is an object
  • Simple and Secure: Easy to learn with built-in security features
  • Robust: Strong memory management and error handling
  • Multithreaded: Supports concurrent programming

Java Ecosystem

Core Components

  • JDK: Java Development Kit
  • JRE: Java Runtime Environment
  • JVM: Java Virtual Machine

Popular Frameworks

  • Spring Boot: Web applications
  • Hibernate: Database operations
  • Maven/Gradle: Build tools

Why Java? Java is one of the most popular programming languages in the world, used by millions of developers for building enterprise applications, web services, mobile apps, and more.

Hello World Code Explanation

Complete Source Code

/**
 * The most basic Java Hello World application
 * This is a simple console application that prints "Hello, World!" to the terminal
 */
public class HelloWorld {
    
    /**
     * The main method - entry point of any Java application
     * @param args command line arguments (not used in this simple example)
     */
    public static void main(String[] args) {
        // Print Hello World message to the console
        System.out.println("Hello, World!");
        System.out.println("This is a minimal Java application running locally.");
        System.out.println("Java version: " + System.getProperty("java.version"));
        System.out.println("Operating System: " + System.getProperty("os.name"));
        System.out.println("User: " + System.getProperty("user.name"));
    }
}

Code Breakdown

Code Element Purpose Explanation
/** ... */ Documentation comment Javadoc comment explaining the class purpose
public class HelloWorld Class declaration Defines a public class named HelloWorld
public static void main Main method Entry point where program execution begins
String[] args Command line arguments Array to receive command line parameters
System.out.println() Print statement Outputs text to the console
System.getProperty() System information Retrieves system properties like Java version

Key Java Concepts

Important Points

  • Class Name = File Name: HelloWorld.java must contain class HelloWorld
  • main() Method: Every Java application needs a main method
  • public static void: Required signature for the main method
  • Case Sensitive: Java is case-sensitive (HelloWorld ‚â† helloworld)

Compilation Process

Java Compilation Steps

Java follows a two-step process: compilation and execution. Let's understand each step:

Step-by-Step Compilation

1Write Source Code: Create HelloWorld.java

2Compile to Bytecode: javac HelloWorld.java ‚Üí HelloWorld.class

3Execute Bytecode: java HelloWorld

Compilation Command

# Compile the Java source file
javac HelloWorld.java

# This creates HelloWorld.class file

What Happens During Compilation?

Stage Input Process Output
Source Code HelloWorld.java Human-readable Java code Text file with .java extension
Compilation HelloWorld.java javac compiler converts to bytecode HelloWorld.class
Execution HelloWorld.class JVM interprets bytecode Program output

Understanding Bytecode

Bytecode is an intermediate form of code that's platform-independent. The JVM on any operating system can execute Java bytecode, which is why Java is "write once, run anywhere."

Common Compilation Errors

  • Syntax Errors: Missing semicolons, brackets
  • Class Name Mismatch: File name doesn't match class name
  • Missing main Method: No entry point defined
  • Import Issues: Missing or incorrect imports

Running the Application Locally

Execution Methods

There are several ways to run our Java application locally:

Method 1: Using Our Run Script

# Make the script executable
chmod +x run.sh

# Run the script
./run.sh

Method 2: Manual Compilation and Execution

# Step 1: Compile the Java file
javac HelloWorld.java

# Step 2: Run the compiled class
java HelloWorld

Method 3: One-liner (if Java is in PATH)

# Compile and run in one command
javac HelloWorld.java && java HelloWorld

Our Run Script Explained

#!/bin/bash
# Set JAVA_HOME to the Java installation
export JAVA_HOME="/opt/homebrew/Cellar/openjdk/24.0.1/libexec/openjdk.jdk/Contents/Home"
export PATH="$JAVA_HOME/bin:$PATH"

# Check if Java is installed
if ! command -v java &> /dev/null; then
    echo "‚ùå Java is not installed."
    exit 1
fi

# Compile and run
javac HelloWorld.java && java HelloWorld

What Happens When We Run?

Execution Flow

1JVM Startup: Java Virtual Machine initializes

2Class Loading: HelloWorld.class is loaded into memory

3main() Execution: The main method is called

4Output Generation: System.out.println() statements execute

5Program Termination: JVM shuts down

Local Execution: Unlike web applications that run on servers and are accessed via browsers, our Java application runs directly on your local machine and outputs to the terminal.

Java Basics Explained

Core Java Concepts

1. Classes and Objects

public class HelloWorld {  // Class definition
    // Class contains methods and variables
}

Class: A blueprint or template for creating objects. In our case, HelloWorld is a class.

2. Methods

public static void main(String[] args) {  // Method definition
    // Method body contains executable code
}

Method: A block of code that performs a specific task. The main method is the entry point.

3. Access Modifiers

  • public: Accessible from anywhere
  • private: Accessible only within the same class
  • protected: Accessible within package and subclasses
  • default: Accessible within the same package

4. Static Keyword

static: Means the method belongs to the class itself, not to any instance. You can call static methods without creating an object of the class.

5. Data Types

Type Example Description
String "Hello, World!" Text data
int 42 Integer numbers
double 3.14 Decimal numbers
boolean true/false True or false values

6. System.out.println()

Breaking Down System.out.println()

  • System: Built-in Java class
  • out: Static field representing standard output
  • println(): Method to print a line of text
