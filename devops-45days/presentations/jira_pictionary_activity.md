## ğŸ® **How to Play**

### **Round Structure:**
1. **Each round:** 90 seconds
2. **Teams rotate:** Drawer changes each round
3. **Scoring:** Points based on difficulty level
4. **Total rounds:** 6-7 rounds (depending on time)

### **Rules:**
- âœ… **CAN DO:**
  - Draw pictures, symbols, diagrams
  - Use arrows and lines
  - Draw numbers (for story points, etc.)
  - Use gestures while drawing

- âŒ **CANNOT DO:**
  - Write letters or words
  - Speak or make sounds
  - Use pre-agreed signals
  - Point to objects in the room

### **Special Jira Drawing Tips:**
- **Bug:** Draw a literal bug/insect
- **Epic:** Draw a big book or mountain
- **Sprint:** Draw a running person
- **Burndown Chart:** Draw a declining graph
- **Swimlane:** Draw swimming lanes in a pool
- **Workflow:** Draw arrows connecting boxes
- **Backlog:** Draw a pile of papers/tasks
