Manually Configured

This web server was set up manually after launching the EC2 instance. This approach gives you complete control over the installation process and allows you to customize each step according to your specific requirements.

Setup Process:
  1. Launched basic EC2 instance
  2. Connected via SSH
  3. Updated system packages
  4. Manually installed Apache HTTP Server
  5. Started and enabled Apache service
  6. Created this custom web page
  7. Configured proper file permissions

Both automated (User Data) and manual approaches have their advantages. The manual approach is excellent for learning, troubleshooting, and situations where you need fine-grained control over the setup process.

Benefits of Manual Setup:
• Better understanding of each step
• Easier debugging and troubleshooting
• Full control over the installation process
• Ability to customize on-the-fly
• Great for learning and development
