ğŸ”§ Core Spring Boot Components

ğŸ“± Main Application Class

@SpringBootApplication public class HelloWorldApplication { public static void main(String[] args) { SpringApplication.run(HelloWorldApplication.class, args); } }

@SpringBootApplication Annotation

  • @Configuration: Tags the class as a source of bean definitions
  • @EnableAutoConfiguration: Enables Spring Boot's auto-configuration
  • @ComponentScan: Enables component scanning in the current package

ğŸŒ Web Controller

@Controller public class HelloWorldController { @GetMapping("/") public String home(@RequestParam(defaultValue = "World") String name, Model model) { model.addAttribute("greeting", "Hello, " + name + "!"); return "index"; } @GetMapping("/api/hello") @ResponseBody public String helloApi(@RequestParam(defaultValue = "World") String name) { return "{\"message\": \"Hello, " + name + "!\"}"; } }

Key Annotations Explained

  • @Controller: Marks class as a web controller
  • @GetMapping: Maps HTTP GET requests to handler methods
  • @RequestParam: Binds request parameters to method parameters
  • @ResponseBody: Indicates return value should be written directly to HTTP response
